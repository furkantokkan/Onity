# Onity

Onity is one Unity package that unifies dependency injection, reactive programming, and events behind a single coherent API — replacing the usual mix of Zenject/VContainer, R3/UniRx, and MessagePipe. The core is engine-free; ZLinq is the only third-party runtime dependency.
